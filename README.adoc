* With the increase in the number of insurance plans handle by a particular insurance agent, he/she will find that
 managing client's data and profile are getting increasingly difficult, thus the need for reInsurance
* reInsurance is a desktop application that serves as an client tracking book for insurance agents to use to manage information about their clients.
* reInsurance provides revolutionary technology that not only aims to organize the clutter of the agent's address book
but also to simplify tedious tasks of calculating important information.

== PROJECT: NUSCouples

---

== Overview

NUSCouples is a CLI-based desktop app targeted towards couples studying in the National University of Singapore (NUS).
Written in Java and boasting a GUI client created with JavaFX, it has a total of about 10 kLoC.

== Summary of contributions

* *Major enhancement*: added *a motivate command*
** What it does: allows the user to send a motivational picture to their partner with just one command/one click.
** Justification: this feature allows the user to stay afloat and develop a positive mood amidst a hectic school schedule.

* *Minor enhancement*: added aliases for commands.

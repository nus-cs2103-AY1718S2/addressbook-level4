AddSurance is a comprehensive, reliable and convenient address book for insurance agents built on a simple belief -- efficiency and elegance.
With its minimalistic design, user interactions are greatly simplified using a CLI (Command Line Interface). With this application, you can
maintain your huge list of contacts easily, at the same time, managing your schedule efficiently.
So, if you are a reasonably tech-savvy insurance agent, this application is absolutely designed for you!

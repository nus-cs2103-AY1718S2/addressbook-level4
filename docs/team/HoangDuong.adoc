=== New feature: Auto-completion
==== Motivation

Without auto-completion feature, users need to spend lots of their precious time just to type long commands and think of what missing fields they need to complete their commands. Auto-completion helps users save their time and effort so that they can enjoy interesting features of the application.

These features help users manage their recipes more efficiently. It will be harder for users to manage their recipes when they have too many recipes. Allowing users to create custom groups of recipes provides a more convenient and flexible method for them to manage.
